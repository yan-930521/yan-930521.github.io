### 主旨
簡約風格的2d橫向格鬥遊戲
### 按鍵控制

### 技能列表
#### Sprint - 衝刺
> 在120毫秒內，移動6次，每次移動10個像素的距離
- 拖曳著 黃 -> 綠 -> 藍 的殘影特效
- 移動期間，透明度下降

#### BlueFireBall - 幽藍爆炎
> 續力一秒後，在100毫秒內，總共10次，每次都給火球一個0.5大小的力
- 藍色粒子特效
